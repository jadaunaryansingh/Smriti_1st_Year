# Video Generation Prompt for Mega Platform

## App Overview
Create a video showcasing "Mega Platform" - a modern, comprehensive learning and competition platform for developers, problem-solvers, and innovators. The app features a sleek dark theme with vibrant gradient accents and glassmorphism effects.

## Visual Design & Theme

### Color Scheme
- **Background**: Deep dark slate (slate-950, slate-900) with gradient overlays
- **Primary Color**: Vibrant purple/violet gradient (HSL: 264 85% 58%)
- **Secondary Color**: Bright cyan/blue gradient (HSL: 199 89% 50%)
- **Accent Color**: Golden yellow (HSL: 48 96% 56%)
- **Text**: White and light slate tones with gradient text effects
- **Borders**: Subtle slate-700/800 borders with glow effects on hover

### Visual Effects
- Glassmorphism: Frosted glass effects with backdrop blur
- Gradient overlays: Animated pulsing gradient orbs in background
- Hover animations: Scale transforms, glow effects, color transitions
- Smooth transitions: All interactions have fluid 300ms transitions
- Shadow effects: Multi-layered shadows with colored glows (primary/secondary)

## User Interactions & Animations

### Hover Effects
- Buttons: Scale up (105%), shadow glow, opacity changes
- Cards: Border color change, background gradient overlay, icon scale (125%), text gradient effect
- Links: Color transitions, gap expansion, arrow translation
- Social icons: Scale up (125%) on hover

### Background Animations
- Pulsing gradient orbs: Slow pulse animation with blur
- Gradient overlays: Subtle opacity changes
- Card glows: Blur intensity increases on hover

### Transitions
- All elements: Smooth 300ms transitions
- Transform effects: Scale, translate, rotate
- Color transitions: Gradient shifts, opacity changes

## Responsive Design
- **Mobile**: Single column layouts, stacked buttons, hidden navigation items
- **Tablet**: 2-column grids, adjusted spacing
- **Desktop**: 3-4 column grids, full navigation, optimal spacing

## Typography
- **Font**: Inter (Google Fonts) - weights 400, 600, 700, 800
- **Headings**: Bold/black weights with gradient text effects
- **Body**: Regular weight, slate colors
- **Sizes**: Responsive scaling from mobile to desktop
